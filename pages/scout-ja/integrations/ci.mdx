# 継続的インテグレーションで Docker Scout を使用する

Docker Scout CLI プラグインや GitHub Action を使用して、Dockerイメージをビルドする際に継続的インテグレーションパイプライン内で分析することができます。

利用可能な連携:
- GitHub Actions
- GitLab
- Microsoft Azure DevOps Pipelines
- Circle CI
- Jenkins

また、CI/CDパイプラインの一環としてランタイム連携を追加し、デプロイ時にイメージを `production` や `staging` といった環境に割り当てることもできます。詳細については [Environment モニタリング](https://chatgpt.com/environment/_index.md)をご覧ください。
