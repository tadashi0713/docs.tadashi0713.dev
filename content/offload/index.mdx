# Docker Offload

Docker Offload は、既存の Docker ツールを使ってコンテナのビルドや実行をクラウドにオフロードできるフルマネージドサービスです。
これにより、クラウドインフラを活用して 高速かつ一貫性のあるビルド や、LLM の実行や機械学習パイプラインといった 計算負荷の高いワークロード を効率的に処理できます。

このドキュメントでは、Docker Offload の概要、セットアップ方法、ワークフローでの利用法、そして一般的な問題のトラブルシュートについて解説します。
