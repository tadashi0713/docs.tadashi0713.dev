# フィードバックを送る

Docker Offload に関するフィードバックを提供する方法はいくつかあります。

## クイックサーベイ

最も手早く意見を共有する方法は、この短い [Docker Offload フィードバックサーベイ](https://docker.qualtrics.com/jfe/form/SV_br8Ki4CCdqeIYl0) に回答することです。
1 分程度で完了し、Docker チームがユーザー体験を改善するのに役立ちます。

## 製品内でのフィードバック

Docker Desktop ダッシュボードの各ビューには **Give feedback** リンクがあります。
ここからフィードバックフォームを開き、Docker チームに直接アイデアを共有できます。

## GitHub でバグや問題を報告する

バグや問題を報告するには、以下のページを参照してください：

- [Mac 版 Docker Desktop の問題 (GitHub)](https://github.com/docker/for-mac/issues)

- [Windows 版 Docker Desktop の問題 (GitHub)](https://github.com/docker/for-win/issues)

- [Linux 版 Docker Desktop の問題 (GitHub)](https://github.com/docker/desktop-linux/issues)
