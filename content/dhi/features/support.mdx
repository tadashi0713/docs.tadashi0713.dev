# エンタープライズサポート

Docker Hardened Images（DHI）は、エンタープライズ環境における柔軟性と強力なサポートを提供するよう設計されており、各チームが自社のニーズに合わせてイメージを調整しつつ、セキュリティとコンプライアンスを確保できるようになっています。

## エンタープライズグレードのサポートと SLA に基づくセキュリティアップデート

Docker は DHI ユーザー向けに包括的なエンタープライズサポートを提供しており、セキュリティ脅威や運用上の問題に迅速に対応します:

- エンタープライズサポート: Docker のサポートチームにアクセスでき、ミッションクリティカルなアプリケーションの保護と運用継続性を重視した応答体制が整備されています。

- SLA に基づく CVE 対応: Docker は、重大（Critical）および高（High）深刻度の脆弱性（CVE）について、アップストリームで修正が公開されてから原則7営業日以内に対応することを目標としています（一部例外あり）。この対応は業界標準より迅速であり、エンタープライズレベルの SLA によって保証されているため、チームはワークロードを安全に保つためのタイムリーな修正に安心して依存できます。

このサポート体制により、組織は DHI をミッションクリティカルなアプリケーションに安心して利用でき、セキュリティと安定性が事前に確保されているという信頼を得ることができます。
