# 特長

Docker Hardened Images（DHI）は、単なるミニマルなベースイメージやアプリケーションイメージを超えて、ハードニングされた初期設定、署名付きメタデータ、幅広いエコシステムとの互換性を備えています。

単一のサービスを保護する場合でも、組織全体でコンプライアンス管理を導入する場合でも、このセクションでは DHI を本番環境に適した選択肢にしている主要な特長を紹介します。
