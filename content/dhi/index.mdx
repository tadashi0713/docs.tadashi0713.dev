# Docker Hardened Images

Docker Hardened Images（DHI）は、Docker が管理する、最小構成でセキュアかつ本番環境に対応したコンテナベースイメージおよびアプリケーションイメージです。
脆弱性を削減しコンプライアンス対応を簡素化することを目的に設計されており、DHI は既存の Docker ベースのワークフローに、ほとんどまたはまったくツールの変更なしで統合できます。

以下のセクションでは、Docker Hardened Images の利用を開始する方法、ワークフローへの統合方法、そして DHI がセキュアかつエンタープライズ対応である理由について学ぶことができます。
