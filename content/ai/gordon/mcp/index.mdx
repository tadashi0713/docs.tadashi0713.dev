# Model Context Protocol (MCP)

[Model Context Protocol](https://modelcontextprotocol.io/introduction) （MCP）は、アプリケーションが大規模言語モデル（LLM）にコンテキストや追加機能を提供する方法を標準化するオープンプロトコルです。
MCP はクライアント・サーバープロトコルとして機能し、たとえば Gordon のようなアプリケーション（クライアント）がリクエストを送信し、サーバーがそのリクエストを処理して AI に必要なコンテキストを提供します。
このコンテキストは、MCP サーバーがアクションを実行するコードを実行して結果を取得したり、外部 API を呼び出したり、その他の類似した処理によって収集される場合があります。

Gordon は、Claude Desktop や Cursor などの他の MCP クライアントと同様に、コンテナとして実行されている MCP サーバーと相互作用できます。
